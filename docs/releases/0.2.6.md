# Release Notes: 0.2.6

## Highlights

This release polishes the shared social icon treatment in the theme package. The header icons now breathe more cleanly, and the footer keeps a quieter variant without relying on incidental outer-scope link styling.

## Major Themes

### Social icon styling cleanup

- added explicit social icon group spacing inside the shared social menu component
- gave header icons a calmer default cold-silver blue tone with a brighter hover state
- kept footer icons more muted instead of forcing header/footer into one unified chrome treatment

### Shared component ownership

- moved icon tone and spacing ownership into `SocialMenu.astro`
- passed explicit header/footer presentation intent from the shared header and footer components
- removed dependence on parent selectors trying to reach through the social menu component boundary

## Migration Notes

- existing users can upgrade by updating `@anglefeint/astro-theme`; no starter or content migration is required
- starter sync is still required for this release so new starter users receive the same package version and matching shared component output

## Starter Sync

- starter sync is required for this release
