Welcome to everyone trying Anglefeint.

If you are reading this, you are now part of a small but serious builder community that values visual identity, engineering quality, and practical iteration speed.

Theme repository:

- https://github.com/anglefeint/astro-theme-anglefeint

What we are building is not just another blog skin. Anglefeint is designed as a route-based cinematic system: Matrix on home, neon cyberpunk for the archive, AI control-room for post detail, and hacker terminal for About.

We also build this project in an AI-native workflow. Different tools help at different layers:

- Codex: architecture refactors, code edits, release automation, and reproducible engineering tasks.
- Cursor: editor-centered implementation flow, rapid file navigation, and day-to-day development speed.
- Claude Code: review perspective, alternate implementation critique, and quality challenges.
- Other AI coding assistants are adopted as needed, but only when they fit our engineering constraints and review flow.

These tools do not replace engineering judgment. They amplify it when used with clear constraints, tests, and release discipline.

So this is our public welcome note:

- use the theme,
- break it,
- improve it,
- and make your own signature style with it.

Thanks for building with us.
